## Описание программы
Book Tracker — это простое настольное приложение для ведения личной коллекции прочитанных книг. Программа позволяет добавлять, удалять и фильтровать книги, а также сохранять данные в JSON-файл.

## Возможности программы
- Добавление книги (название, автор, жанр, количество страниц)
- Просмотр всех книг в виде таблицы
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по количеству страниц (например, больше 200)
- Удаление выбранной книги
- Автоматическое сохранение данных в `books.json`
- Загрузка данных из JSON при запуске

## Проверка корректности ввода
- **Название, автор, жанр:** не могут быть пустыми
- **Количество страниц:** должно быть целым положительным числом
- При попытке добавить книгу с пустыми полями программа выдаст сообщение об ошибке
- При указании некорректного числа страниц (буквы, символы, пустая строка) программа отклонит добавление

## Требования
- Python 3.6 или выше
- Стандартная библиотека Python (модули `tkinter`, `json`, `os`)

## Установка и запуск
1. Скачайте файл `book_tracker.py`
2. Скачайте или создайте пустой файл `books.json` (программа создаст его сама при первом сохранении)
3. Поместите оба файла в одну папку
4. Убедитесь, что Python установлен правильно (команда `python --version` в терминале)

## Запуск программы
Откройте терминал/командную строку в папке с программой и выполните:
```bash
